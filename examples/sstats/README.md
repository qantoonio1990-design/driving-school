# Пример работы с SStats.net Football API

## Что это за ресурс

[**SStats.net**](https://sstats.net) — сервис футбольной статистики с **бесплатным REST API**.
Личный кабинет находится по адресу [`https://sstats.net/profile`](https://sstats.net/profile):
там пользователь видит своё имя и **персональный API-ключ**, который используется
для авторизации запросов и повышенных лимитов.

Назначение API — программно получать футбольные данные в формате **JSON/CSV**:

- список лиг, команд, игроков, сезонов и турнирных таблиц;
- расписание и результаты матчей (с фильтрами по лиге, сезону, дате, «сегодня»);
- детальная статистика матча, события, составы, травмы;
- доматчевые и **live**-коэффициенты букмекеров и их история;
- аналитика: рейтинги **Glicko 2**, **xG** (ожидаемые голы) и вероятности исходов.

## Доступ и авторизация

- Базовый URL: `https://api.sstats.net`
- Авторизация: query-параметр `?apikey=<ВАШ_КЛЮЧ>`
- Лимиты: без ключа — 300 запросов/мин суммарно и 30/мин на один IP; с ключом лимит выше.
- Спецификация OpenAPI 3.1: <https://sstats.net/openapi/v1.json>
- Документация и быстрый старт: <https://sstats.net/api>

## Основные эндпоинты

| Метод/Путь                  | Назначение                                  |
| --------------------------- | ------------------------------------------- |
| `GET /Account/Info`         | Данные аккаунта (проверка ключа)            |
| `GET /Leagues`              | Список лиг с сезонами                        |
| `GET /Games/list`           | Список матчей (`?today=true`, `?LeagueId=`, `?Year=`) |
| `GET /Games/{id}`           | Детальные данные матча                       |
| `GET /Games/glicko/{id}`    | Прогноз Glicko 2 + xG + вероятности исходов  |
| `POST /Games/query`         | Расширенный поиск матчей                      |
| `GET /Odds/{gameId}`        | Доматчевые коэффициенты                       |
| `GET /Odds/live/{gameId}`   | Live-коэффициенты                            |
| `GET /Teams/{id}`           | Информация о команде                          |
| `GET /Players/{id}`         | Данные игрока                                 |

## Запуск примера

Требуется Node.js 18+ (используется встроенный `fetch`).

```bash
# ключ берётся из личного кабинета https://sstats.net/profile
export SSTATS_API_KEY=ваш_ключ
node sstats-example.mjs
```

Скрипт по шагам демонстрирует: проверку аккаунта, список лиг, матчи за сегодня,
детали конкретного матча и прогноз Glicko 2 / xG.

## Пример запроса напрямую

```bash
curl "https://api.sstats.net/Games/list?today=true&apikey=ВАШ_КЛЮЧ"
curl "https://api.sstats.net/Games/glicko/1183255?apikey=ВАШ_КЛЮЧ"
```
